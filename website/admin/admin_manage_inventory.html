Gérer les stocks

Utilisez ce panneau pour mettre à jour les niveaux de stock. L'ajout de stock ici déclenchera automatiquement la génération des passeports produits et les notifications de retour en stock.

Inventaire des Produits
Modifier le Stock Total

ATTENTION !

Vous êtes sur le point de **modifier manuellement le stock total**. Cette action **ne créera PAS de passeports produits**.

Utilisez cette fonction uniquement pour des corrections d'inventaire (ex: après un comptage physique). Pour ajouter de nouveaux articles avec des passeports, utilisez la fonction "Ajouter du Stock" sur la page principale.
